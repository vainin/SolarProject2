Description


These images came from an image AI. They show just how useful using an AI to create images is.
The AI that was used to create these images was Craiyon. It is a free way to create images.
How to use the AI is type what you want an image created of in the prompt then click the draw button and you will be given plenty of images to choose out of.
A prompt that can be used is an animal name such as “Corgi” or something silly like “fat cat sitting on chocolate.”
